# Dependencies

In order to function to have to addtionally setup a device-control server.

Read more about it at: https://github.com/nuriofernandez/devices-control

# How to install?

Create a symbolic link or place the folder `smart-lights@nurio.me` at the following path:

```
~/.local/share/gnome-shell/extensions/smart-lights@nurio.me
```

Copy `settings.json.sample` as `settings.js` and set your `device-control` server credentials.
